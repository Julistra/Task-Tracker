### Conventions
- **IDs**: `{PREFIX}.{NNN}` persist forever; do not renumber.
- **Quadrant**: UI / IN / UN / NN
- **Status**: NEW / NEXT / WAIT / DOING / DONE / DEFER / DROP
- **Dates**: YYYY MMM DD

### Format within tasks-file for each Domain

#### Domain of Tasks: human name

> Full task list for this Domain (active tasks and completed tasks). Each task is block keyed by its ID.

#### Active

##### {PREFIX.NNN}
- **Task**: Short imperative label
- **Quadrant**: UI | IN | UN | NN
- **Status**: NEW | NEXT | WAIT | DOING | DONE | DEFER | DROP
- **Due**: YYYY-MM-DD (or `—`)
- **Tags**: #tag1 #tag2
- **Links**: [Doc/Folder](URL)  
- **Notes**: Context, next step, owner, constraints.
